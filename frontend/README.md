# Ecommerce Frontend

React + Redux frontend for a full-stack ecommerce application.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| UI | React 17, React Bootstrap, Bootstrap 5 |
| State | Redux Toolkit, Redux Thunk |
| Routing | React Router v6 |
| HTTP | Axios |
| Payments | Stripe |
| SEO | React Helmet |

## Getting Started

```bash
npm install
npm start
```

App runs at `http://localhost:3000`.

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Development server |
| `npm run build` | Production build |
| `npm test` | Run tests |

## Project Structure

```
src/
  components/   # Reusable UI components
  pages/        # Route-level pages
  store/        # Redux slices and store config
```
